# 快速启动指南

## 启动前准备

确保已安装：
- Docker 和 Docker Compose
- Python 3.8+
- Node.js 16+
- PostgreSQL 数据库（包含 job_postings 表）

## 1. 启动 Neo4j

```bash
docker-compose -f docker-compose-neo4j-only.yml up -d
```

## 2. 启动后端服务

**终端1 - 启动主应用：**
```bash
cd backend
python full_app.py
```

**终端2 - 启动图谱服务：**
```bash
cd backend
python job_skill_graph_service.py
```

## 3. 启动前端服务

```bash
cd frontend
npm run dev
```

## 4. 访问应用

- 前端：http://localhost:5173
- 主应用 API：http://localhost:8082
- 图谱 API：http://localhost:7576/health
- Neo4j 浏览器：http://localhost:7474 (neo4j/password123)

## 停止服务

```bash
# 停止 Neo4j
docker-compose -f docker-compose-neo4j-only.yml stop

# 停止后端和前端：Ctrl+C 在对应终端（需要停止2个后端进程）
```

## 完整重启

```bash
# 停止所有服务
docker-compose -f docker-compose-neo4j-only.yml down

# 重新启动
docker-compose -f docker-compose-neo4j-only.yml up -d
cd backend && python job_skill_graph_service.py
# 新开终端
cd frontend && npm run dev
```
